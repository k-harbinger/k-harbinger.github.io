Turtle Project

About the Project

This project is about creating a graphical scene using Python's Turtle graphics module. I used the provided building block functions (draw_rectangle, draw_square, draw_circle, etc.) to create composite shapes such as a moon, stars, and a castle. I altered the existing draw_triangle function to be focused on a point radially so that I could create 6-pointed stars.

Sample Code

This function creates a 6-pointed star using my custom equilateral triangle function. 


def draw_star(t, size, fill_color):
    """draw star"""
    x, y = t.pos()
    draw_eqtri(t, x, y, size, rotation_angle=0, fill_color=fill_color)
    draw_eqtri(t, x, y, size, rotation_angle=60, fill_color=fill_color)
