Turtle Project

About the Project

This project is about creating a graphical scene using Python's Turtle graphics module. I used the provided building block functions (draw_rectangle, draw_square, draw_circle, etc.) to create composite shapes such as a moon, stars, and a castle. I altered the existing draw_triangle function to be focused on a point radially so that I could create 6-pointed stars.

Sample Code

This function creates an equilateral triangle which will be re-used to create a 6-pointed star.

def draw_eqtri(t, center_x, center_y, size, rotation_angle, fill_color): """Draw an equilateral triangle centered at a point with a rotation angle""" t.penup() # Calculate the 3 vertices relative to the center points = [] for i in range(3): angle_deg = rotation_angle + i * 120 angle_rad = math.radians(angle_deg) x = center_x + size * math.cos(angle_rad) y = center_y + size * math.sin(angle_rad) points.append((x, y)) # Move to first point and start fill t.goto(points[0]) t.pendown() t.fillcolor(fill_color) t.begin_fill() # Draw lines to each vertex for point in points[1:] + [points[0]]: t.goto(point) t.end_fill() t.penup() t.goto(center_x, center_y) t.pendown() def draw_star(t, size, fill_color): """draw star""" x, y = t.pos() draw_eqtri(t, x, y, size, rotation_angle=0, fill_color=fill_color) draw_eqtri(t, x, y, size, rotation_angle=60, fill_color=fill_color)
